Descripción: Este proyecto es un gestor de ventas con fines educativos, desarrollado como parte de un ejercicio práctico.
Características
Gestión básica de ventas.
Registro de interacciones.
Enfoque en prácticas de desarrollo de software educativo.
Requisitos Previos
Asegúrate de tener instalado Python en tu sistema.
Instalación
Clona este repositorio en tu máquina local:
git clone https://github.com

Instala las dependencias necesarias usando pip y el archivo requirements.txt:
pip install -r requirements.txt

Uso
Para ejecutar el bot, simplemente corre el archivo principal Auto_3.py desde tu terminal:
python Auto_3.py

Contribuciones
Las contribuciones son bienvenidas. Siéntete libre de abrir issues o enviar pull requests.
